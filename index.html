HTTP Security Headers

A Quick Introduction

Adam Hart   ::   Sygnul, LLC   ::   sygnul.com

Hi, my name is Adam Hart. I work on Sygnul, which is an unreleased, end-to-end encrypted collaboration app. I wanted to give a brief overview of some of the HTTP headers built into modern browsers that deal with security.

HTTP Headers


HTTP/1.1 200 OK
Accept-Ranges: bytes
Cache-Control: max-age=604800
Content-Type: text/html
Date: Mon, 27 Jul 2015 00:05:46 GMT
Etag: "359670651"
Expires: Mon, 27 Jul 2015 00:05:46 GMT
Last-Modified: Fri, 09 Aug 2013 23:54:35 GMT
Server: ECS (ftw/FB85)
X-Cache: HIT
x-ec-custom-error: 1
Content-Length: 1270
Headers from example.com
Every request the browser makes has meta-data associated with it such as cache controls, content types, and any custom ones you throw in there.
